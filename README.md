# WhatsApp UI Clone

A simple and responsive WhatsApp-style chat interface built using HTML and CSS.
This project recreates the basic WhatsApp web layout with a chat list panel, chat window, message bubbles, and input area.

---

## 🚀 Features

* Responsive WhatsApp-style layout
* Chat list sidebar
* Chat window for messages
* User avatar placeholders
* Message bubbles (sent & received)
* Input field with send button
* Clean green and white WhatsApp-inspired theme

---

## 🛠️ Technologies Used

* HTML5
* CSS3
